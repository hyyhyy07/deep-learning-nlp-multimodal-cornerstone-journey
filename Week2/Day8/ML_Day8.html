Day 8 - MLP - 从逻辑回归到神经网络¶

目标：理解"神经元 = 逻辑回归 + 激活函数"，多层堆叠 = MLP

Week 2 第一天：从经典 ML 一脚踏入深度学习

关键洞察：逻辑回归就是一个神经元¶

逻辑回归:  z = w.x + b  ->  Sigmoid(z)  ->  p
神经元:    z = w.x + b  ->  f(z)         ->  a   (激活值)

区别只在于：逻辑回归固定用 Sigmoid，神经元可以用 ReLU/Tanh 等

Day 8 - MLP 多层感知器
========================================

1. 激活函数三巨头 - 给网络加"弯"¶

没有激活函数 -> 多层 = 一层（线性变换的复合还是线性） 有了激活函数 -> 可以拟合任意曲线

函数 公式 输出范围 特点 用在哪
Sigmoid 1/(1+e^{-z}) (0, 1) 两端梯度接近 0 二分类输出层
Tanh (e^z-e^{-z})/(e^z+e^{-z}) (-1, 1) 零中心 RNN
ReLU max(0, z) [0, +inf) 计算快、梯度不消失 隐藏层默认 
Sigmoid: z 很大或很小时梯度≈0 -> 梯度消失
Tanh:   比 Sigmoid 好（零中心），但仍有梯度消失
ReLU:   z>0 时梯度=1（不消失!），z<0 时梯度=0（神经元死亡）
-> 隐藏层默认用 ReLU，输出层二分类用 Sigmoid

2. XOR 问题 - 为什么单层不够？¶

逻辑回归 = 一条直线分割 -> 半月形数据直线切不开 加一层隐藏层 -> 曲线出现了

单层只能线性分割，多层可以非线性分割

单层只能画直线，多层可以画曲线

3. MLP 结构 - 层层递进¶

输入层       隐藏层       输出层
  x1 ---+
        +--> h1 ---+
  x2 ---+          +--> y
        +--> h2 ---+
  x3 ---+

h = f(W1 @ x + b1)           # 隐藏层: 线性变换 + 激活
y_hat = sigmoid(W2 @ h + b2) # 输出层: 线性变换 + Sigmoid

输入: [1. 2. 3.]
z1 = W1@x + b1:      [ 0.216  0.035  0.171 -0.178]
h = ReLU(z1):        [0.216 0.035 0.171 0.   ]
z2 = W2@h + b2:      [-0.031]
y_pred = Sigmoid(z2): 0.4923

W1: (4, 3), W2: (1, 4)
参数总数: 21

4. 隐藏层大小实验 - 神经元越多越好吗？¶

和决策树的 max_depth 一样：太小欠拟合，太大过拟合

最佳: (2), 测试准确率=0.982456
最大: (50,50), 测试准确率=0.973684

5. 激活函数对比 - ReLU vs Tanh vs Sigmoid¶

同一个网络，不同激活函数，效果差多少？

隐藏层 -> ReLU | 二分类输出 -> Sigmoid | 多分类输出 -> Softmax

Day 8 完成！¶

概念 一句话
神经元 逻辑回归 + 激活函数 -> z=w.x+b, a=f(z)
ReLU max(0,z)，隐藏层默认，梯度不消失
Sigmoid 二分类输出层，输出概率
前向传播 h=f(W1@x+b1), y=f(W2@h+b2)
MLP > 线性 加隐藏层 -> 非线性决策边界
