Day 16 - LSTM - 长短期记忆网络¶

目标: 理解 LSTM 如何用三个门解决 RNN 的梯度消失问题

核心: 细胞状态 C_t (信息高速公路) + 三个门 (遗忘/输入/输出)

RNN vs LSTM¶

RNN:   h_t = tanh(W_xh * x_t + W_hh * h_{t-1})
       只有一个隐藏状态, 梯度连乘 -> 消失

LSTM:  C_t = f_t * C_{t-1} + i_t * C~_t    (细胞状态, 加法!)
       h_t = o_t * tanh(C_t)               (隐藏状态)
       加法传递梯度 -> 梯度不会消失!

RNN 的梯度路径:   连乘 (W_hh^T)^{T-1}  → 指数衰减
LSTM 的梯度路径:  加法 C_t = f_t * C_{t-1} + ...  → 梯度直通!

1. LSTM 的三个门¶

LSTM 的关键创新: 细胞状态 C_t (Cell State) —— 一条贯穿所有时间步的信息高速公路

遗忘门 (Forget Gate) - 决定扔掉什么¶

f_t = sigmoid(W_f * [h_{t-1}, x_t] + b_f)
输出 0~1, 乘到 C_{t-1} 上
  1 = 完全保留这段记忆
  0 = 完全丢弃这段记忆

输入门 (Input Gate) - 决定写入什么¶

i_t = sigmoid(W_i * [h_{t-1}, x_t] + b_i)    ← 写入哪些位置
C~_t = tanh(W_C * [h_{t-1}, x_t] + b_C)      ← 要写入的新信息

输出门 (Output Gate) - 决定输出什么¶

o_t = sigmoid(W_o * [h_{t-1}, x_t] + b_o)    ← 输出哪些位置
h_t = o_t * tanh(C_t)                          ← 输出门调控隐藏状态

完整流程¶

   C_{t-1}                 C_t
      |                     ^
      |   遗忘  +   输入    |
      |    f_t  *  C_{t-1}  +  i_t * C~_t
      v                     |
  [x_t, h_{t-1}] -> 三个门 -> o_t -> h_t

只有加法! C_t = f_t * C_{t-1} + i_t * C~_t
加法路径的梯度 = 1, 不会消失!

输入形状: (5, 4)
隐藏状态形状: (5, 8)
最后时间步 h[:8]: [-0.0539 -0.0022 -0.0175 -0.0531 -0.0188  0.0157  0.0154  0.0373]

2. 梯度消失对比实验: RNN vs LSTM¶

RNN 的梯度经过连乘, LSTM 的梯度经过加法 + 遗忘门

实验: 给两个网络相同的输入, 观察 x1 对后面 h_t 的影响

RNN:   dh_t/dh_{t-1} = W_hh^T * tanh'(h_{t-1})      ← 连乘, 必然消失
LSTM:  dC_t/dC_{t-1} = f_t                            ← 遗忘门控制的加法!
       如果 f_t ≈ 1, 梯度几乎无损传递

3. PyTorch nn.LSTM 使用¶

用法跟 nn.RNN 完全一致, 但多返回一个细胞状态 C

nn.LSTM(input_size, hidden_size, num_layers, batch_first, dropout, bidirectional)

输出:
  output: (batch, seq_len, hidden_size * num_directions)
  (hn, cn): (num_layers * num_directions, batch, hidden_size)
            ↑ 隐藏状态          ↑ 细胞状态

input:  (4, 7, 10)
output: (4, 7, 20)
hn:     (2, 4, 20)  ← 隐藏状态
cn:     (2, 4, 20)  ← 细胞状态
output[:,-1] == hn[-1]: True

今天学到了什么¶

概念 一句话
细胞状态 C_t LSTM 的"高速公路", 梯度通过加法直通, 不会消失
遗忘门 f_t sigmoid 输出 0~1, 控制扔掉多少旧记忆
输入门 i_t sigmoid 输出 0~1, 控制写入多少新信息
输出门 o_t 控制展示多少细胞状态给隐藏状态
LSTM > RNN 加法路径 -> 梯度不消失 -> 长记忆更好
4 倍参数 LSTM 有 4 组权重 (i/f/o/g), RNN 只有 1 组

LSTM 梯度分析¶

RNN:   dh_t/dh_{t-1} = W_hh^T * tanh'(h_{t-1})
       连乘 T 次 -> 指数级消失

LSTM:  dC_t/dC_{t-1} = f_t
       如果 f_t ≈ 1:  梯度无损! 记忆可以传 1000 步
       如果 f_t ≈ 0:  丢弃旧记忆, 重置
       网络可以学习 f_t 的值 -> 自主决定什么时候遗忘!
